What have I learned from this class?

What I have learned in this class is to understand how to create a web page, how to follow a design pattern and create styles using CSS. Also to use already created styles such as those in W3.CSS. Thanks to this course I have understood more deeply about the importance of having a code that is clean of errors and semantically well structured. I was also able to learn the history of my avatar when I chose her, I had no idea who she was.

I really think that HTML and CSS is a basic course for every future programmer because it helps you stay organized when coding. This course will definitely help me when I want to create a website in the future. It has been an excellent course since I put my creativity on each page.
